User's can follow specific Tags or category and show only those tags/category posts.

== Description ==
This plugin will easily give the user the option to follow the category or tag of his choice. This is basically a widget based plugin, it will work from the time you activate your tag / category follow widget after you install the plugin. One of the conditions for this to work is that the user must be logged in.

== Installation ==
1. Upload the 'follow-tags-categories' folder to the '/wp-content/plugins/' directory.
2. Activate the 'Follow Tags & Categories' plugin through the 'Plugins' menu in WordPress.
3. Add and set Widget from Appearance->Widget
